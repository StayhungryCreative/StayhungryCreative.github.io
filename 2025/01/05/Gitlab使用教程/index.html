Gitlab使用教程

2025-01-05 工具 gitlab, ubuntu 评论

按照 git 常用的操作来记录

1、git config 配置本机信息

2、git init 将本地当前目录初始化为Git仓库

3、git add 将本地文件到缓存区

4、git commit 提交本地更改

5、git remote 添加远程Git仓库

6、git push 将本地Git仓库的更新推送到远程Git仓库
